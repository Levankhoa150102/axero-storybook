# Drawer Component

A sliding panel for navigation, management, or contextual content. Appears from the right, overlays the page, and can be closed by clicking the overlay or the close button.

## Features
- Appears from the right or left side (configurable via `placement` prop)
- Overlay darkens the background
- Customizable title and content
- Close by overlay or button
- Responsive and accessible

## CSS Classes
- `.drawer-root` — Main container
- `.drawer-open` — Active state
- `.drawer-overlay` — Background overlay
- `.drawer-panel` — Drawer panel
- `.drawer-header` — Header area
- `.drawer-title` — Title text
- `.drawer-close` — Close button
- `.drawer-content` — Content area
- `.drawer-footer` — Footer area

### Conditional Classes

The root container uses conditional classes based on the open state:

- When the drawer is **closed**: `<div class="drawer-root">`
- When the drawer is **open**: `<div class="drawer-root drawer-open">`

This ensures the correct styles and transitions are applied depending on whether the drawer is visible.

## Use Cases
- Admin panels
- Contextual navigation
- Management menus
- Support/help drawers

## Placement

The Drawer can appear from either the right or left side of the screen. Change the class `drawer-panel` to control this:

- To the right: `<div class="drawer-panel drawer-panel-right">`
- To the left: `<div class="drawer-panel drawer-panel-left">`
