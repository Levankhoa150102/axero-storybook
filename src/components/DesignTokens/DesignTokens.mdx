# Design Tokens

Design tokens are the visual design atoms of the Axero Design System. They are named entities that store visual design attributes like colors, typography, spacing, and effects. These tokens ensure consistency across all components and make theming and customization easier.

## How to Use

All design tokens are available as CSS custom properties (variables) and can be used in your CSS:

```css
.my-component {
  background-color: var(--background-primary);
  color: var(--body-text-primary);
  border: 1px solid var(--border-color-primary);
  border-radius: var(--widget-border-radius);
  box-shadow: var(--box-shadow-common);
}
```
