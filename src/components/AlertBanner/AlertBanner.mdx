# Alert Banner Component

The Alert Banner component displays temporary notification messages to users. It supports different message types, auto-hide functionality, and generates raw HTML for AlpineJS integration.

## AlertBanner Types

The component supports four different AlertBanner types:

- **Success** (`type="success"`) - Green background for positive feedback
- **Warning** (`type="warning"`) - Yellow background for warnings
- **Error** (`type="error"`) - Red background for errors
- **Info** (`type="info"`) - Blue background for informational messages


## CSS Classes

The component uses the following CSS classes:

- `#message-box-wrapper` - Fixed positioned container
- `.message-box` - Table layout for centering
- `.message-content` - Main content container
- `.message-content--success` - Success variant styling
- `.message-content--warning` - Warning variant styling
- `.message-content--error` - Error variant styling
- `.message-content--info` - Info variant styling
- `.message-content-inner` - Message text container
- `.message-content-options` - Close button container
- `.close-link` - Close button styling

# Notification Component

The Notification component displays persistent banners for alerts, info, success, warning, and error messages. It supports a main message, optional description, and action button.

## Notification Types
- **Info** (`type="info"`)
- **Success** (`type="success"`)
- **Warning** (`type="warning"`)
